E-Shagrid

Project information

  • Category: Web Development
  • Project date: 1 Feb, 2021

E-Shagrid is a Location Based Tutor Finding Application. The Purpose of this application is to provide students an excellent tutor at there current location based upon reviews added by older students on their courses.

Developing using React JS, Material UI, TypeScript, Node JS , Nest JS, PostgreSQL, Docker, Prisma and React Google Maps API.
